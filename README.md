# dotfiles

## dependencies

- git
- hyper

## getting started

1. Install Dependencies

2. Clone this repository into your home directory
  ```
  cd ~
  git clone https://github.com/Ryomm/dotfiles.git
  ```

3. Change the directory and set symlink
  ```
  ln -s ~/dotfiles/.hyper.js ~
  ln -s ~/dotfiles/.gitconfig ~
  ```

4. Run the scripts
  ```
  ./setup_brew.sh
  ```
